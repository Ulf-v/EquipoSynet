### Tecnologías tratadas (Discusiones iniciales)
- **Frontend:** Debate entre React vs Vue vs JavaScript vanilla
  - React: Componentización, pero requiere más tiempo de aprendizaje
  - Vue: Intermedio en complejidad
  - JavaScript vanilla: Simple pero menos mantenible a largo plazo
  - **Decisión aplazada** hasta tener más claridad de requisitos
  
- **Backend:** Java Spring Boot mencionado como opción principal
  - Estabilidad y documentación
  - Compatible con requisitos hospitalarios
  
- **Base de datos:** MongoDB como opción principal
  - Flexibilidad con documentos
  - Escalabilidad horizontal
  
- **Infraestructura:** Docker mencionado como necesario para despliegue
