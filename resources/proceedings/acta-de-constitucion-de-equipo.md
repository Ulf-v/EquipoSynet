## 3. Objetivo general del proyecto
Desarrollar una plataforma web compuesta por:

- **Backend** basado en **Java + Spring Boot**, conectado a una base de datos **MongoDB** alojada en un servidor dedicado.  
- **Frontend** desarrollado en **React**, con diseño apoyado en **Bootstrap**.  
- Integración de **Google Auth** para el sistema de autenticación.  
- Contenedorización y despliegue mediante **Docker**.

El objetivo final es entregar un sistema funcional, modular, escalable y correctamente documentado.

---

## 4. Alcance inicial
El proyecto contempla:

- Creación de un backend funcional con endpoints definidos y documentados.  
- Desarrollo de un frontend modular, responsive y conectado al backend mediante API.  
- Implementación de autenticación mediante Google Auth.  
- Configuración y acceso remoto a la base de datos MongoDB Server instalada en el servidor asignado.  
- Pruebas de integración entre frontend, backend y base de datos.  
- Uso de datos de prueba cargados en MongoDB para validar el sistema.  
- Gestión del proyecto mediante herramientas colaborativas.
